## NodeJS

- An environment to run JavaScript outside the browser.
- It's built on Chrome's V8 JS engine(every browser has an engine).

### Pre-requisite
- HTML, CSS, JavaScipt, ES6
- Callbacks, Promises, Async-Await
- General understanding of JavaScript

### Content
- Intro
- Installation
- Node Fundamentals
- Express Tutorial
- Building Apps

### Differences between Browser and NodeJS

Browser -

- DOM
- Window
- Interactive Apps
- No Filesystem
- Fragmentation
- ES6 Modules

NodeJS -

- No DOM
- No Window
- Server Side Apps
- Filesystem
- Versions
- CommonJS(for module)
